## developer setup:
> install pyenv, virtualenv
>
> pyenv virtualenv 3.9.4 wordle
>
> pyenv activate wordle
>
> pip install --upgrade pip
>
> pip install -r requirements.txt
>
> add discord token to .env file
>
> .env file contents: DISCORD_TOKEN={your token}
